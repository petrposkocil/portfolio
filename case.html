: LULC Classification

Supervised classification

in remote sensing is a method of automatically categorizing pixels in a satellite or aerial image into different land cover or land use classes based on known training samples. It involves defining a set of sample classes, each with a known land cover or land use type, and using this information to train a classifier. The classifier is then applied to the entire image, using the training data to make predictions about the land cover or land use of each pixel in the image.

: MCDA

Multi-Criteria Decision Analysis (MCDA) is a systematic process for evaluating multiple options based on a set of criteria. The process involved clearly defining the decision problem and identifying the relevant criteria, which in this case were Urban areas, Water, Slope, Schools, Stadiums, and Transport. This data was then normalized to make it comparable, which was done by converting it into a standard format. The importance of each criteria was then assigned based on its significance to the decision problem. Final result was calculated using map algebra.

Heatmap (kernel density)

A geospatial heatmap is a type of heatmap that is used to display data values on a geographical map. It is a way to represent the spatial distribution of data values by using different colors to represent different ranges of values. The heatmap provides a graphical representation of data values by using different colors to represent different ranges of values.

Raster normalization

is a process used to standardize the values of a raster dataset. Normalization is applied to adjust the range of values in the raster data to a standard scale, making it easier to compare and perform mathematical operations with different raster datasets. In raster normalization, the values of a raster image are transformed so that they fall within a specific range, such as 0 to 1 or -1 to 1. This is done by subtracting the minimum value of the raster dataset from each cell, and then dividing by the range, which is calculated as the difference between the maximum and minimum values.

Map algebra

is a tool in geographic information systems (GIS) that allows for the integration of multiple data layers to make decisions. In a MCDA, map algebra is used to analyze multiple criteria, such as accessibility, water sources, and slopes, and to make a final decision on the best alternative. The process involves normalizing and weighting the criteria, combining them into a single map, evaluating alternatives, making a decision, and monitoring it. Map algebra makes the MCDA process more efficient and consistent.
